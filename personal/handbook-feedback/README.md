# Handbook Feedback

学习中遇到的 Handbook（https://aiweb3.school/zh/handbook/）问题、卡点、错别字、概念不清楚、资料过期、结构建议——都记到这里。

## 命名

`YYYY-MM-DD-<short-slug>.md`，例如 `2026-05-26-rag-context-window.md`。

## 必填 4 字段（启动 prompt 要求）

每个 feedback 文件用下面这个结构，缺一不可，方便后续被索引/分析：

```markdown
---
date: YYYY-MM-DD
handbook_url: https://aiweb3.school/zh/handbook/xxx
type: typo | unclear | outdated | structure | broken-link | missing-content | other
severity: low | medium | high
---
```

## 回流路径（按严重程度）

| 严重度 | 推荐回流方式 |
| --- | --- |
| low（错别字、链接坏）| 直接在 [aiweb3school GitHub repo](https://github.com/lxdao-official/aiweb3school) 开 issue 或 PR |
| medium（概念不清、举例不当）| 先在本目录沉淀 2–3 条，再一起开 issue |
| high（章节错 / 资料过期 / 链接全死）| 立即开 issue + 在 TG 群里 @ nanako |
